# 🏅 Olympic Medal Prediction

Machine learning project focused on predicting Olympic medal outcomes using historical Olympic data and socio-economic indicators.

The project investigates which factors are most strongly associated with Olympic success and compares the predictive performance of multiple machine learning models.

---

## 🎯 Project Objectives

- Predict Olympic medal counts for different countries
- Identify key factors associated with Olympic success
- Compare classification and regression approaches
- Evaluate model performance on real-world sports data

---

## 📊 Dataset

The analysis combines historical Olympic Games data with country-level socio-economic indicators.

Examples of variables include:

- Population
- GDP
- GDP per capita
- Number of athletes
- Number of events participated in
- Historical Olympic performance
- Country characteristics

---

## 🔍 Data Analysis

The project includes:

- Data cleaning and preprocessing
- Missing value handling
- Exploratory Data Analysis (EDA)
- Feature engineering
- Correlation analysis
- Variable selection

---

## 🤖 Machine Learning Models

### Regression Models

- Linear Regression
- Ridge Regression
- Lasso Regression

### Classification Models

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM)
- Random Forest
- Neural Networks

---

## 📈 Model Evaluation

Models were compared using:

### Regression Metrics

- RMSE
- MAE
- R²

### Classification Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
